# unity 延迟渲染

## G-Buffer

在 unity 延迟渲染中, unity 要求我们输出 4 个 G-Buffer. 分别是

| RT   | 描述                               |
| ---- | ---------------------------------- |
| RT0  | rgb 反照率, a 遮蔽                 |
| RT1  | rgb 高光(菲涅尔因子) a 光滑度      |
| RT2  | rgb 法线 (N * 0.5 + 0.5), a 未使用 |
| RT3  | rgb 自发光 + 简介光                |
